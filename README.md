# Team_ML10
Team Project
### Methods & Technologies (Maryam)

**Methods:**  
- Explore client demographics, financial status, past campaigns, and macroeconomic     factors.  
- Measure campaign performance with conversion rates.  
- Compare conversion rates by age, job, marital status, loan status, and number of calls.  
- Build predictive models: Logistic Regression, Decision Trees, Random Forest.  
- Identify key factors influencing subscriptions and explain predictions with SHAP.  
- Analyze diminishing returns from repeated calls.  
- Study economic impacts on subscription demand.  
- Create visualizations and dashboards for insights.  

**Technologies:**  
- Python: pandas, numpy, matplotlib, seaborn, scikit-learn  
- Jupyter Notebook for workflow and experimentation  
- GitHub for version control  
- Markdown for documentation  
- SQL for structured queries if needed  
- Visualization apps for dashboards and interactive charts
